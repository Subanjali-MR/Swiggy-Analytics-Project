# Swiggy Data Analytics Project (SQL + Excel Dashboard)

## Project Overview
This project analyzes Swiggy food delivery data using SQL Server and Microsoft Excel Dashboard to generate business insights related to orders, restaurants, locations, and customer ratings.

## Tools Used
- SQL Server (Database Design and Querying)
- Microsoft Excel (Dashboard and Visualization)

## Excel Dashboard Highlights
- Interactive Dashboard Design
- Pivot Tables and Pivot Charts
- KPI Metrics Analysis
- Monthly Order Trends
- Restaurant Performance Analysis
- Category-wise Sales Analysis

## SQL Project Highlights
- Data Cleaning and Validation
- Null Value Checking
- Duplicate Data Removal using ROW_NUMBER()
- Star Schema Data Modeling
- Created Dimension Tables:
  - Date
  - Location
  - Restaurant
  - Category
  - Dish
- Fact Table Creation with Foreign Keys
- Data Insertion using JOIN operations

## Files Included
- Swiggy Database.sql → Complete SQL Project
- Swiggy Excel Dashboard.xlsx → Interactive Excel Dashboard

## Key Insights
- Identified top-performing restaurants and locations.
- Analyzed customer ratings and order trends.
- Monthly and category-wise revenue patterns.
